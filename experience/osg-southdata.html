Projects Managed

Project Sandbag

Designed and developed a web-based data entry system to address critical accuracy issues in the production prep stage at SouthData's Mobile, AL imaging facility. The legacy manual measurement process was prone to human error, impacting downstream production metrics. The solution integrated HID-compliant digital scales with a streamlined interface that captures box weight, employee identification, work order assignment, and precise timestamps. Built on ASP.NET 4.0 with SQL Server backend, the system implements real-time validation rules and intuitive operator workflows. Beta testing demonstrated a 93.26% reduction in measurement standard deviation, fundamentally transforming prep stage data reliability.

ASP.NET Microsoft SQL Server HID Integration Real-Time Validation Process Optimization

Project Symphony

Engineered a comprehensive business intelligence reporting system to provide granular visibility into production facility operations. The platform features Employee Bio Sheets for individual performance tracking and Work Order Reports for project-level analysis, powered by SQL Server data warehousing and ASP.NET visualization dashboards. The system captures detailed metrics including task duration, images processed, and complexity factors across all production stages—prepping, scanning, and quality control. By integrating previously siloed data from customer service interactions and jacketing logs, Symphony dramatically reduced unaccounted labor time. The reporting accuracy reached 90% for labor cost attribution, enabling data-driven profitability analysis and operational decision-making.

ASP.NET Microsoft SQL Server Data Analysis KPI Metric Development Process Optimization
